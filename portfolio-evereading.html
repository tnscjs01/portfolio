portfolio detail

  • Thymeleaf와 Spring Data JPA를 Spring MVC와 함께 결합해 하나의 테이블을 이용하는 방명록 생성
  • 목록 : 전체 목록을 페이징 처리해서 조회할 수 있고, 제목/내용/작성자 항목으로 검색과 페이징 처리
  • 등록 : 새로운 글을 등록할 수 있고, 등록 처리 후 다시 목록 화면으로 이동
  • 조회 : 목록 화면에서 특정한 글을 선택하면 자동으로 조회 화면으로 이동, 조회 화면에서는 수정/삭제가 가능한 화면으로 버튼을 클릭해서 이동
  • 수정/삭제 : 수정 화면에서 삭제가 가능하고 삭제 후에는 목록 페이지로 이동, 글을 수정하는 경우에는 다시 조회 화면으로 이동해서 수정된 내용 확인
