Jakarta EE, 구 Java EE(Java Enterprise Edition / J2EE)

기업에서 운영하는 서버 페이지에 특화된 에디션이다. JSP와 서블릿을 비롯한 웹 애플리케이션 서버에 관련된 기술들이 포함되어 있다.

Java ME(Java Micro Edition / J2ME)
