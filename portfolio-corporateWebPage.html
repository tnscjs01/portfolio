portfolio detail

  • 시맨틱 태그와 CSS를 활용한 기본 기업형 웹 페이지
  • 시맨틱 태그는 웹 페이지의 영역을 나눠서 웹 브라우저가 문서를 효율적으로 읽을 수 있도록 도와줌
  • <header>태그 안에는 웹 페이지의 상단 영역인 메뉴를 그룹화
  • <figure>태그 안에는 <video> 태그를 활용한 비주얼 영역 제작
  • 가운데 <section>태그에는 콘텐츠 영역 구성
  • 마지막 <footer>태그에는 하단 영역으로 기업 정보 입력
